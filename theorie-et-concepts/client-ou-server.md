# Client ou server

### Backend

_JavaScript côte serveur_ étend ces éléments de base avec des objets utiles pour le fonctionnement sur un serveur tels que la possibilité de <mark style="color:orange;">communiquer avec une base de données</mark>, <mark style="color:orange;">manipuler des fichiers</mark>, passer d'une application à une autre, etc.

### Frontend

_JavaScript côté client_ étend ces éléments de base en fournissant des objets pour <mark style="color:orange;">contrôler le navigateur et le</mark> <mark style="color:orange;"></mark>_<mark style="color:orange;">Document Object Model</mark>_ <mark style="color:orange;"></mark><mark style="color:orange;">(DOM)</mark>. Par exemple, les extensions du langage côté client permettent de placer des éléments dans un formulaire HTML, de <mark style="color:orange;">réagir aux événements</mark> déclenchés par l'utilisateur (les clics, la saisie d'un formulaire, les actions de navigation, etc.).
