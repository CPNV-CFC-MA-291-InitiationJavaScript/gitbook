# Expérience Navigateur

## Objectifs

* [ ] Expérimenter l'exécution d'un script directement au sein d'un navigateur
* [ ] Se familiariser avec les outils d'analyse de votre navigateur
* [ ] Modifier du code directement dans le navigateur

## Todo

Par équipe "rangée" de 4 développeurs

* [ ] Rangée 1 -> installez et testez la console JavaScript sur Chrome
* [ ] Rangée 2 -> installez et testez la console JavaScript sur Firefox
* [ ] Rangée 3 -> installez et testez la console JavaScript sur Safari
* [ ] Rangée 4 -> qu'est-ce que le mode "strict" en javascript ?
  * [ ] [https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Strict\_mode](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Strict_mode)

## Livrables

Il s'agit ensuite de présenter à la classe le résultat de votre expérimentation afin que chacun puisse ensuite profiter de vos recherches.

Un document doit synthétiser vos recherches. Il vous servira à présenter et démontrer vos conclusions et découvertes.
