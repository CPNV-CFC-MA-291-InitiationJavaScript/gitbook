# Evaluations

### Evaluation I

#### Description

Le candidat connaît les bases du langage nécessaire à la réalisation d'un projet simple. Il réalise un projet de développement simple à l'aide de son environnement de développement.\
\
Il est également capable de lire du code produit par un autre développeur et d'expliquer le comportement de celui-ci, sans avoir besoin de l'exécuter.\
\
Il démontre aussi la capacité à utiliser des outils de debug et de gestions d'erreur.

| Critère               | Description                                                                                                                                                                                                                                                                                                                                       |
| --------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Moyens auxiliaires    | <p>Un résumé manuscrit d'une page A4 (recto). Le code produit durant les exercices.<br>Aucun accès réseau.</p>                                                                                                                                                                                                                                    |
| Forme d'examen        | Ecrite et pratique                                                                                                                                                                                                                                                                                                                                |
| Forme collaborative   | Individuelle                                                                                                                                                                                                                                                                                                                                      |
| Critères d'évaluation | <p>*Généralités*<br>Répondre correctement à des questions théoriques, avec et sans code et réussir un petit projet à l'aide de son environnement de développement.<br><br>* le vocabulaire de base est acqui.<br>* le code livré s'exécute.<br>* l'environnement de développement est maîtrisé pour coder, exécuter et debuger l'application.</p> |
| Pondération           | 1/3 de la note du module                                                                                                                                                                                                                                                                                                                          |
| Durée                 | 2 périodes                                                                                                                                                                                                                                                                                                                                        |
